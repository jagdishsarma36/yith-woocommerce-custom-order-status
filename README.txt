YITH WooCommerce Custom Order Status allows you to create and manage custom statuses for orders in WooCommerce.


== Description ==
Create unlimited statuses for your WooCommerce orders and make your "Orders" section less chaotic with YITH WooCommerce Custom Order Status.
A brand-new plugin by YIThemes that helps you improve WooCommerce features in order management. This plugin allows you to make "Orders" page much more intuitive, immediate and easier to use. It puts at your disposal customisable labels to highlight and differentiate different steps of your users' checkout process and to have an immediate overview on orders in your shop.
= Features: =

* A specific panel to create unlimited order statuses. Options allow to set a different label and color for each status.
* Possibility to assign custom statuses to orders.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder and upload it into the directory `wp-content/plugins/` in your WordPress site.
3. Activate `YITH WooCommerce Custom Order Status` from Plugins page

YITH WooCommerce Custom Order Status will add a new submenu called "Custom Order Status" under "YIT Plugins" menu. Here you are able to configure all the plugin settings.
